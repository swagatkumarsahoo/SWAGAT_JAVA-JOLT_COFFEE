# SWAGAT COFFEE CAFE

## Overview

Welcome to **SWAGAT COFFEE CAFE**, a static website for Swagat's Java Jolt Coffee, a charming cafe located in Ameerpet, Hyderabad, Telangana, India. This project showcases the cafe's story, menu, team, customer reviews, and provides a reservation system for table bookings. The site is designed to be responsive, visually appealing, and easy to navigate, capturing the warm and inviting vibe of a local coffee shop.

The website is built as a static HTML/CSS/JavaScript application, making it lightweight and fast to load. It features a modern design using Bootstrap 5 for responsiveness, custom CSS for branding, and JavaScript libraries for interactivity like slideshows and smooth scrolling.

Live Preview: https://tourmaline-praline-038a68.netlify.app/

## Features

- **Homepage (index.html)**:

  - Hero section with a dynamic slideshow background.
  - About section with an embedded video and cafe history.
  - Team section introducing the staff (Swagat, Sandra, Jackson, Michelle).
  - Menu section divided into Breakfast and Coffee categories with prices and descriptions.
  - Reviews section with customer testimonials and ratings.
  - Contact section with a form, embedded Google Map, and cafe details.
  - Footer with location, contact information, social links, and opening hours.

- **Reservation Page (reservation.html)**:

  - Dedicated booking form for table reservations.
  - Fields for full name, phone number, time, date, number of people, and optional message.
  - Responsive design matching the homepage theme.

- **Responsive Design**: Fully responsive across devices (desktop, tablet, mobile) using Bootstrap 5.
- **Interactive Elements**: Smooth scrolling navigation, slideshow effects, and form validation.
- **Media Integration**: Includes images, videos, and icons for an immersive experience.

## Technologies Used

- **HTML5**: Structure and content of the website.
- **CSS3**: Custom styling with Bootstrap 5 framework for layout and responsiveness.
- **JavaScript**: Interactivity using jQuery, Bootstrap JS, and Vegas.js for slideshows.
- **Bootstrap 5**: Front-end framework for responsive grid, components, and utilities.
- **Vegas.js**: Background slideshow plugin.
- **Google Fonts**: Plus Jakarta Sans for typography.
- **Bootstrap Icons**: Icon library for social links and UI elements.

## Installation and Setup

Since this is a static website, no server-side setup or dependencies are required. Follow these steps to run the project locally:

1. **Clone the Repository**:

   ```
   git clone https://github.com/yourusername/SWAGAT_COFFEE_CAFE.git
   cd SWAGAT_COFFEE_CAFE
   ```

2. **Open in Browser**:

   - Simply open `index.html` in your preferred web browser (e.g., Chrome, Firefox).
   - For the reservation page, open `reservation.html`.

3. **Optional: Serve Locally**:
   - If you prefer a local server, use tools like Live Server (VS Code extension) or Python's built-in server:
     ```
     python -m http.server 8000
     ```
     Then navigate to `http://localhost:8000/index.html`.

No additional installations are needed as all assets are included.

## Usage

- **Navigation**: Use the navbar to jump to sections like Home, About, Menu, Reviews, and Contact.
- **Reservation**: Click the "Reservation" button on the homepage to access the booking form.
- **Contact**: Fill out the contact form or use the provided email/phone for inquiries.
- **Social Links**: Connect via Facebook, Twitter (X), and WhatsApp from the footer.

The site is optimized for performance and accessibility, with semantic HTML and ARIA labels where applicable.
